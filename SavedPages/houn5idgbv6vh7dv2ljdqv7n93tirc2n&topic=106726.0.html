Looks nice. I need some more red in my key sets. I will probably pick up an ergo set and maybe the novelties and a deskmat. Also happy to see the anime inspired themes, especially KonoSuba.

I mentioned this late in the IC and at this point it is probably too late to change even if you wanted to, but why is the Caps Lock icon arrow with a line pointing down? KAT/KAM is the only key cap style I have ever seen use this symbol that way. All other uses of this icon outside of KAT/KAM have the arrow pointing up with the line below it. Most other sets that don't use the underlined shift arrow use a lock symbol, and a handful of others (some GMK and JTK sets, as well as Ducky and some Razer keyboards) use a down arrow (inverted shift icon) with no line. Using an inverted shift arrow is strange to me, but I could see an argument for it being more flexible to be used as Function or maybe even a Control key and it doesn't appear to be entirely uncommon. The the bar over the down arrow is what puzzles me. Is this a Keyreative manufacturing thing, or did you specifically design it that way?
