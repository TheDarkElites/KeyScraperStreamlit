Update 2017/5/4 :
Silent Red is not enough...
When we packing the switches.
We found too many people order silent red which is much more than IC form...
They're not enough even we packed all the switches we plan to sell for retail on our store.
So some of orders include Silent red will delay
The queue is FIFO depends on order date.

The other issues is we made a mistake on mounting type of clear.
We ordered PCB mount but list plate mount T_T
We've send mail to guys who ordered switches
Please check your mail, if you didn't receive it, please send us message.

Update 2017/5/1 :
Thank you guys.
This GB has ended and we've started shipping this week.
But just get info from Cherry representative that Linear Gray deliver date will delay.
So orders include Linear Gray has to ship later.
Sorry for that

GB link here : https://shopkey.doyustudio.com/005swcgb
FB link of this GB : https://www.facebook.com/douystudio/posts/789562617869841

We're going to order some switches.
And some of them is going to put on this GB, 120 pcs each pack.
Include :
1. RGB Cherry silver ($46, 0.383/pc)
2. Non RGB, plate mounted silence red.($48, 0.4/pc)
3. Non RGB, plate mounted Clear (not sure now, 53, 0.44/pc)
4. Non RGB, plate mounted Linear Gray ($40, 0.333/pc)

We made an IC for this GB.
Most people interested in Silver and Silent Red.
But not many people interested in Clear and Linear Gray.
We'll order them anyway, the quantity might limited.
I guess IC not only reflect trend on Geekhack, but also on the market of all people -0-.
We don't want to stock too many items that might not able to sell out for over 1 year.

This GB will end on 4/30 23:59, 2017 (UTC +8)
And start to ship from May 1, 2017.
Ship by small pack, DGM, air pack or DHL.

Since we've ran another Cherry switches GB around Dec 2016.
So we only provide types that do not show in last one.

If shipping fee is > $10 for single pack, mean there might has some error.
Please contact us, include quantity, country in the message to help us spot what cause the issue
If you have any question, leave comment here.
Or send us a mail : sy_yang@doyustudio.com, sales@doyustudio.com, or send message to Facebook page.


We received some feedback about other switches.
Like silent black, PCB mount RGB...etc
But the quantity in too low for us
For example, it hard for us to sell 2 cases of plate mounted MX Black each year.(16000 pcs)
But, the quotation we get is ask us for 20000 pcs MoQ.
It's way too far compare to the MoQ if we order those switches(Maybe only less than 10 people ask for each type 0.0).
This also means they're hard to sell on the market, so we have to deal with horrible stock if we run the GB for those switches.
We only order some to sell in our store for higher price, and the price might be too high to run a GB.
Anyway, we might do some IC later for those switches.
But in my opinion, possibility to run the GB is not high
Sorry for those who are looking for switches we didn't provide.
