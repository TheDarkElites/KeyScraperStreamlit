Colors






More pictures of the original version here.

The Standard Edition boards (Back to the Future, 8-Bit, Hard Candy and Night Drive) do not include front cover cutouts for blocker lighting. However, DIY blocker RGB lighting hardware (LEDs, individual PCB, etc.) is included in all variants regardless of cutouts. Special RGB Edition boards (Laser and Mech Love) include blocker RGB cutouts by default. All variants come with "LESS IS MORE" engraved above the USB Type-C port. In addition, a distinctive retro aging effect is applied exclusively to Mech Love, with a unique finish on each individual board.


PCB

1.2mm 4-layer hot-swap split backspace PCB with in-switch LEDs and cutouts. USB Type-C (supports C to C) and Bluetooth 5.1 connectivity. Supports wireless charging. Additional PCBs for Touch Panel and blocker RGB lighting built-in. RGB and key layout configurable via Angry Miao DIY site. All keys support remapping and assigning multimedia keys (except the Fn+Esc Power On combo), supports up to 7 layers.


Plate

POM plate compatible with split backspace.


Included with each keyboard

  • Stainless steel leaf-spring with mount (4pcs)
  • Phosphorus copper leaf-spring with mount (2pcs)
  • T6 screwdriver
  • T4 L-key screwdriver
  • T6 torx screw M2*4 (2pcs)
  • T6 torx screw (PVD) with CD thread M2*9 (2pcs)
  • T4 torx screw with CD thread M2*3 (2pcs)
  • Allen cap screw M2*4 (2pcs)
  • Leaf-spring stopper (2pcs)
  • Silicone stands (2pcs)
  • Plate double-end stud (2pcs)
  • Flat cable (4pcs)
  • Bottom pad (adjustment pad for Gear 2)
  • Keyboard case
  • Cleaning cloth
  • Storage pouch
  • User manual

Base Kit includes:
Keyboard body, 3 PCBs (main, touch panel and blocker RGB), POM plate, accessories mentioned above.


Specifications

Mounting method: Angry Miao Two-Stage Adjustable Leaf-Spring Mount
Typing angle: 10°
Weight: Base Kit 1.40kg, Bundle 1.47kg
Material and craftmanship: aluminum alloy body with anodized and/or electrophoretic finish
Dimensions: 296.7mm (L) * 119.8mm (W) * 20.4mm (front height without feet) / 39.2mm (rear height without feet)
Versions: Standard Edition and Special RGB Edition, with "LESS IS MORE" engraving above the USB Type-C port for all colorways; A distinctive retro aging effect is applied exclusively to Mech Love, with a unique finish on each individual board*.
