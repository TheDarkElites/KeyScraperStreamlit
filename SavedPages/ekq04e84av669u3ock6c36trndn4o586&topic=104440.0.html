GMK FINER THINGS




ABOUT

GMK Finer Things is a cap set based on a signature colorway well known in the world of luxury.
This set is meant to be simple and bold, a combination that is easy to achieve, hard to master.

VENDORS

US: Cannon Keys
EU: CandyKeys
Asia: iLumKB
Oceania: Daily Clack
CA: Deskhero

This group buy will be active from February 1 to February 29.

ANNOUNCEMENTS
Deskhero has joined the team as out CA vendor!

CruzCtrl cables will be offering custom cable for this set on his site: https://cruzctrl.gg/

Unofficially working with ZealPC in attempts to bring Silent Tealios to be offered some time this month along side this set.
Teal Healios? Thealios? Finer Thealios?

More news on that later can be found here.

Some of the novelties, namely the ring, moped, sunglasses, and perfume, have been modified to be slightly simpler for the sake of manufacturing. While the board renders have not been updated to reflect that, the kit render has. Please refer to the kit render for the final form of these novelties.
