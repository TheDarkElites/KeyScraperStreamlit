Introduction

G'day champions, my name is Jarrod aka guypigeotto. If you haven't seen this set by now, boy have you missed out.
After an unfortunately failed group buy in October 2021, there has been renewed interest in the set due to a certain manufacturer cloning the set earlier this year.
Well, due to this renewed hype, and the overwhelming interest of vendors to bring this back, this set is reborn as GMK Handarbeige+ as is tradition.


Colours and Kitting

GMK Handarbeige features all stock GMK colours, which will be listed below. It is a combination of the U9+L9 beige background, with the GMK Handarbeit+ array as legends. It also includes V4 front print.
