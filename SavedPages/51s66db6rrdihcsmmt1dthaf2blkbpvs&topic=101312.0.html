GMK Handarbeit+ Keycap set

Hey, Guys!
Here I come back again with GMK Handarbeit+ is one of the most interesting keyset.

This is GMK Handarbeit+ Keycap set GB.

Inspired by G81-3000HBD OG Handarbeit keyboard.
Some years ago, G81-3000HBD OG keyboard with various colorful double shot keycap was found by Ruseupi.
Its housing had engrave as 'Handarbeit', and many users have called this keyboard as just "Handarbeit".

About 3 years ago, I designed this set and finally this set come true!
Just join this GB to get legendary keyset!!
