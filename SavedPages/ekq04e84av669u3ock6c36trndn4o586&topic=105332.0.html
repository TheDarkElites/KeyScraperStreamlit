Hi Everyone,

This is the group buy for this project I have been working for a while now. All the final preparations have been made and the group buy process is now ready.

Introducing the Canon60. This is a hi-profile case comprised of a metal casing encased in acrylic.




I created this keyboard in an attempt to create a floating type design. I also personally like the combination of acrylic and polished metals.
In order to create a clean floating look, there are minimal screws in this design. (Screws are only used to fasten the PCB to the inner case. The rest of the case will be held together with neodymium magnets.

Overall Kit Details:

  • Complete Case Kit w/o PCB = $325.00 (Excludes Shipping)
    • Outer Case: Clear Polished Acrylic
    • Inner Brass Casing:
      • Color Options : Gold PVD, Rose Gold PVD, Black (Electroplated but still highly reflective and glossy
    • Brass PCB Plate:
      • 1.5 mm thick
      • Color Options: Gold PVD, Rose Gold PVD, Black Electroplated
      • PCB plate supports the following layouts
      • Badge (Music Note)
        • Color Options: Gold PVD, Rose Gold PVD, Black Electroplated
      • Angle Steel Blocks
        • Block can provide a 3°, 5°, or 7° depending on which side you choose to mount the block.
        • Gallery of Blocks in Use
      • Adhesive Rubber feet
    • Additional Brass PCB Plate = + $40.00
    • Additional Music Note Badge = + $15.00
    • Additional Angle Blocks = + $10.00
    • Optional 60% PCB: DZ60 by KBDFans = + $38.00
    _____________________________________________________________________________________________________________________________________________________
    Additional Details:

    • Fastening Method: N52 Neodymium magnets will be the main fasteners for this case. The prototype has no problem staying together with these magnets. The magnets are also not so strong that it is too difficult to take the case apart.
      • Color options for neodymium magnets will be silver or gold.
    • Total Weight:
      • About 2.5kg with switches and plate.
      • About 2.1kg for just the case.
    • Case does support any standard 60% tray mount PCB, so you can use your own PCB and plate if a specific layout is not supported.
