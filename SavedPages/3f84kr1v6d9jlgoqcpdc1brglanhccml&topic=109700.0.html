The Colors

Alpha Base: RAL 075 90 20
Alpha Legend: RAL 050 50 60

Mod Base: RAL 350 20 15
Mod Legends: RAL 050 60 60

Accent Base: RAL 050 50 60
Accent Legend: RAL 350 20 15


Note that the alpha legends and the mod legends are two different shades of orange, the reasons for this were twofold, one it adds a bit more diversity in the oranges in the set, and it also helps with the contrast.

I set up a profile for my camera in lightroom using an xrite color checker, but I'm far from an expert. These are in no way guaranteed to be 100% color accurate, but should be used to give an added idea of what the color should be in addition to the renders
