I feel like there are a few things being overlooked in this GB. (It's going to sound negative, so I apologize ahead of time).

Sure the colors are subjective, and honestly, I love how the set looks without the accent. The two-tone would look great, but it doesn't hide the fact that you copied and pasted kitting from your first set without double-checking anything. The only difference in the render is that you moved the extra keys on Numpad next to it from Hanami Dango's render. It seems as the only thing that changed between the two is that you listened to feedback and removed the extra B in the spacebar kit.

As someone pointed out above, this resulted in you keeping an extra pipe key and an extra tilde key when the set clearly does not need it.

As for pricing, this is a mirror of Hanami Dango in regards to color-wise. 3 custom RAL colors, yet somehow, this set is $10 more and $20 more if you want the Hangul sub legends (which I can confirm makes no sense).

Vendors do seem to be much more lenient in buying out MOQ as GMK Matcha Mocha sold more than this set but got canceled back then by the same vendors with a MUCH lower MOQ.
