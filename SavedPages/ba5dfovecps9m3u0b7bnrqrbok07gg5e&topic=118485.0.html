Treasure TYPE-9 ( Series III )

Hi everyone, the GB is now live for the third edition of the Treasure TYPE-9 macropad we are making since 2017.
This new version will come with updated internals and new finish/color options ( IC Thread )

Treasure TYPE-9 is a fully programable metal 9-keys macropad, that can be used for shortcuts, Stream Deck or simply for Artisan display.
QMK & VIA support, compatible with Windows, MacOS and Linux. It also works on iPad ( Works great on Procreate ).
Design-wise, it is a mix of rounded lines and sharp edges, a compact and timeless design that makes it stand out on your desk.







>> JOIN GB HERE <<




Features
  • Crafted from Aluminum 6061 or Solid Brass
  • USB-C interface
  • In-Switch RGB LED
  • 2-piece body construction ( Dedicated Plate and Case )
  • PCB compatible with QMK firmware / VIA
  • Switch Sockets (hot-swappable/no soldering required)
  • Low Profile custom silicone pads with 3M adhesive

Contents
  • 1 x TYPE-9 Case ( Beadblasted Aluminum or PVD Coated + Mirror finish Brass )
  • 1 x TYPE-9 Plate - Bead blasted anodized aluminum ( Grey color )
  • 1 x TYPE-9 Hot Swap PCB ( In-Switch RGB LED)
  • 1 x Treasure USB-C Cable
  • 2 x TYPE-9 Silicone Feets ( Silicone - Grey )





We are proud to present our new finishes : Matte Black, Purple, Silver, Green, Jet Black Mirror (Brass) and Bronze Gold Mirror (Brass), E-White is back.

Solid Brass Special Editions :
Full brass case ( 370gr )
  • Full Solid Brass / Jet Black Mirror ( Jet Black PVD coating + Mirror Polish )
  • Full Solid Brass / Bronze Gold Mirror ( Bronze Gold PVD coating + Mirror Polish )



Aluminum Anodized versions
    Beadblasted Anodized Aluminium ( 150gr )
    • Aluminum / E-White ( Electrophoresis coating )
    • Aluminum / Matte Black
    • Aluminum / Anodized Green
    • Aluminum / Anodized Purple
    • Aluminum / Anodized Silver
