[GB] UM-70 R2| 3D Printed 65% Split Keyboard (Closed, Manufacturing)
« on: Mon, 18 July 2022, 13:40:21 »
UM-70 is a 70 keys split keyboard in 65% layout with RGB, macro keys, OLED screen and encoder housed in PLA 3D printed cases.



FEATURES

IN-SWITCH RGB WITH TRANSLUCENT PANELS


OLED SCREEN


ENCODER


COLOURS (See photos in gb page)
  • Galaxy Purple
  • Gunmetal
  • Orange
  • Blissful Blue
  • Mint
  • White RAL9003
  • Marble Effect

SPECS
  • 70 Keys QWERTY split in 65% layout
  • OLED Screen
  • Encoder (15mm rotary knob)
  • In switch and side penal RGB
  • Top Mounted
  • RF4 Plates
  • 3° typing angle, 4°/6° tented raisers
  • USB-C
  • Hot-swap
  • Powered by QMK, VIA and Vial Compatible
  • PLA FDM 3D Printed Housing
  • Left 170 x 120 x 25mm
  • Right 205 x 120 x 25mm

PCB Layout


SOLDERING IS REQUIRED
This keyboard kit is aimed at tinkers/people who have soldering experience. PCB is partly assembled only. Hot-swap sockets, RGB LEDs, TRRS jack and OLED screen require soldering.

An assembly guide is available here, please check it to see if you're comfortable with the build process before ordering.

PRICE
Self-Print kit £115
Printed kit £145

Ships worldwide

JOIN GROUP BUY HERE

Group buy starts 18th July and ends on 18th August 2022. The estimated shipping date is November 2022. Updates will be posted here, Merge product page and Discord server.


Update:

19/08/22
