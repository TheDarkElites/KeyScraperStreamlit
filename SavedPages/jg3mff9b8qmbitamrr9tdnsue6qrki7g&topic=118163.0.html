Introducing WS Camping

WS Camping applies all-sided dye-sub method. It's a new technique that changes the old dyb-sub heating process and solves problems commonly seen from traditional dye-sub PBT keycaps.

Aurora x Camping Highlights:
  • Variable gasket mount structure. Remove some or all of the gasket chips to change from firm to bouncy. Both Silicone and Poron gaskets are included to fine tune the experience even further!
  • Default PCB (PCB05 Bodo) is a hot-swap PCB that supports multiple layouts with flex cut! This is a very unique feature that lets you use any or all of split L Shift, split R Shift, split Backspace, split spacebar, stepped Caps Lock and ISO Enter, anytime you want. No soldering necessary!
  • The badges are hot-swappable, meaning you can change them without taking apart the case!

More Info about Aurora
Ikki68 Aurora is an affordable 68% variable gasket mount keyboard kit with an injection molded polycarbonate case. The default PCB has Kailh hot swap sockets and supports multiple splits like split space, shifts, backspace and more!
