Welcome to Fusion60 Group Buy.


Fusion60 inspired by 2 of the most well known keyboard case, Fjell and Kira. Design by Vulcan and manufactured by Kbdfans It features a heavy 2.5kg modern 8 degree case, with thick bevels builds from solid aluminum and brass block that are CNC Machined to maintain precision in quality.

F60 comes with a custom made PCB that can form side glow light effect that bleeds from the signature geometrical side cut out. Although it comes with a custom PCB, F60 also supports universal 60% PCB such as  DZ60, 6X, Zeal and many more.

Specifications ;

- 60% (Standard, Winkeyless, HHKB)
- High Profile
- Top Mount
- 8 Degree Angle
- Case Material: Aluminium
- Weight Material: Brass
- Case Finish: Bead Blasted and Anodized
- Plate material: 1.5 mm Brass
- Supports most PCBs
- 2kg

Fusion60 kit Include :

- Aluminum Case
- Brass weight（Sandblasting）
- Brass plate（Sandblasting）
- PCB （QMK Firmware）
- Stabilizers (Cherry Stabilizers )
- Screws

Note :

If you are using the Fusion60 PCB and want to have side glow effect you will need to do this :
- Carefully separate both sides LED PCB from the main PCB
- Solder 4 cable each side of the LED PCB to the main PCB
- Screw both LED PCB to the top case
The reason we do this is because we need to place the LED just a bit higher than the PCB and get it as close as possible to the side to get a bright bleeding effect.
We have already tried many methods but this one works best.

This LED assembly is a bit hard and takes trial and error.. I know it's not the best design, but it's the best i can do to keep the PCB key layout and the price low.
