Milkyway Alpenglow is finally here!!!
Milkyway Alpenglow is a CHERRY profile keycap set by Milkyway




Milkyway Alpenglow was inspired by a spring break trip I had with friends. They offered to take me a on weekday camping trip to cheer me up when I wasn't
doing so well. This set was in honor of the sights I saw during that trip.

Hey, hey, hey!!! R4 Colors have been approved an we have moved onto the QC stage. There have been a few issues with qc, mainly the ones listed below. These issues have been sent to the manu.

Thok Artisans have shipped out.

1) Black needs to be color matched to original black.
2) Row 4 mods need to be colormatched to the original color sample.
3) Row 3 mods need to be matched to the new sample. It seems as if MW has acidentally used the old R3 sample.
4) SB Novs need an SVG update. This has been fixed and a new SVGS have been put into the google drive.
5) SVGS need to be enlarged. New SVGS have been provided for this fix in the google drive.
6) Quality issues with nov clarity. Some of the darker novs seem to have noticebly blurry edges. Row 3 has quite a bit of bleeding/splotching issues.
7) Black+Pink Novs are also not color matched to the rest of the set. All of the sb novs have their color slightly off.

Just a reminder, both desk mats have been approved and have been moved to production.

Also addressing the latest MW issues such as their scratchy caps and qc issues, I will make sure that Alpenglow is not shipped until these qc issues are resolved.

Predicated date of arrival is now 2022 Q4- 2023 Q1
