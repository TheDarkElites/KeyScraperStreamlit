Hello,
We are wind studio.
Thanks for your expectation on wind x98 r2. We decided to start wind x98 r2 on 28th Dec.
More details about wind x98 r2 keyboard kit you can go to check this link:https://geekhack.org/index.php?topic=121998.0



1.Update
The pcb layout of GB production will be updated as this picture. The normal pcb is the same as southpaw pcb, the wired hotswap pcb and bluetooth pcb will include two numpad pcbs,
one support enter left,one support enter right. So southpaw pcb available on bluetooth,hotswap, enter left and enter right layouts at same time.
Support split backspace.



2.Special Note
All parts are the same between normal keyboard and southpaw keyboard, except top upper case. So if you buy normal keyboard and an extra southpaw top upper case, then it can change to a southpaw keyboard.
