British racing green is an iconic colour from the world of automotive sport. Dating back almost 120 years, the colour has been used by Jaguar, Aston Martin, Mini, Lotus and Bentley. After spending a while playing with colours, I settled on RAL 6005, RAL 1001 and GMK's WS1 as the perfect combination for a balanced set within the theme.

GMK British Racing Green Discord: https://discord.gg/XdyjCF5

Kitting

Base Kit



Racing Green Kit



Novelty Kit



Exotic Kit



The base kit has been designed to be inclusive, providing full coverage for 7U/6.25u bottom rows as well as complete UK ISO and numpad support. In addition, I have put together an Exotic kit which provides coverage for some favourite 40s layout, as well as that precious Alice-layout second B key.
