Design Concept:


Inspired from a classic IBM keyboard "PCjr". Named CA66 from French "66% avec de la corniche".



Color Options:
  • Anodized: Gunjyo, Jade
  • Coated: Light Sand, Concrete, Turquoise, Banana
  • Electrophoresis: Milk
Contents:
  • Case
  • Gold color anodized aluminum plate
  • Brass badge
  • Mute foams (one for between plate and pcb, one for back of pcb)
  • Screws and other components
PCB features:
  • Gold pads, south facing LED position
  • USB C interface
  • TMK, QMK firmware compatible. LINK programming software compatible
  • Solderable version & Hotswap version
  • Sorderabe version layout
  • Hotswap version supports 2u Backspace, Normal Capslock, 1u+1.25u+6.25u+1.25u+1u bottom row default layout
BT Hotswap PCB features:
  • Gold pads, south facing LED position, USB C interface
  • USB/Bluetooth BLE 4.2 connections and switch
  • QMK firmware compatible. LINK programming software compatible
  • Layout same as wire connection hotswap version. Split backspace and Iso layout are supported.
  • Compatible with 3.7v Li-ion batteries
Pre-sale time: 2022.5.9 9pm PST
Shipping date: 2022.6
Units: Limited
