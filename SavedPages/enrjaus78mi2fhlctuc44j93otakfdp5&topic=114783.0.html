Space x Cyrillic Cable




This set features new Cyrillic sublegends that Alexbarb and me put together, they are very similar to latin GMK/Cherry in the sense of typeface. The idea was to make as accurate as possible Cyrillic. On top of most noticable legend changes (У, Ж, Л), Cyrillic was completely redone from the ground up. Sublegends on R (К), T (Е), Y (Н), [ (Х), D (В), F (А), H (Р), J (О), Z (Я), C (С), V (M), B (И), N (Т) as well as sublegends on num row will be using original GMK (Cherry) font since glyphs are the same. Rest are custom outlines with original latin legends in mind.

Once GB is over and preparations done on GMK side, this set of legends will be open for everyone to use in their projects. That also includes: modifying particular legends, taking them apart, alternating (for this, you need source files which I can provide upon request), using separate legends in combination with the new legends set.

For the renders with new sublegends please reach out Marco (Marco#8888 on Discord, https://marcofriedrichs.com/).

Huge thanks to Alexbarb for make this happened!

Quote from: konstantin on Fri, 01 October 2021, 04:41:11
GMK doesn't do pad printing.
They do. But if it's not specified, GMK would use UV. The end result of both pretty similar but there is couple of differences such as:
- in order to do multy-color print, you have to stick with UV print, pad print can only handle one color per time
- for UV "primer" (white base) layer need to be applied in order to avoid the color deviation, for the pad on the other side it need to be 2 layers of print because printing layer are too thin and keycap surface can be seen underneath if stick to one layer

Final finish by default are protective layer for both to prevent the erasure of the print.
