Q60 - GB
by Keychron
Discord
Website - order here 

  • Full aluminium body
  • Gasket mount
  • centered HHKB layout
  • Out of the box QMK & VIA support
  • Keychron screw-in stabs
  • 5Pin ANSI hotswap PCB [southfacing]
  • Preinstalled Force case mod/Top break mod [silicon pads between top and bottom case]
  • Physical switch to switch between layers *Explanation down belown
  • Barebones: 179$
  • Fully assembled: 219$



The Q60 is a fully customizable mechanical keyboard that has the unique HHKB 60% layout. The philosophy behind the HHKB layout is it to minimizes any movement from the home row position and gives them complete control of the keyboard and you can still access any function keys through key combinations. The Q60 comes with a complete aluminum CNC machined body, gasket mount + pre installed forcebreak mod, and QMK/VIA support, all designed for a minimal desk setup and premium typing comfort.





*You don't have to use this switch, you can also switch through the layers purely by using VIA, it's just something on top where u can switch between 2 set layers.
As an example:
Switch state 1: VIA layer 1
Switch state 2: VIA layer 3
