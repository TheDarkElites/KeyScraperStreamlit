What?
A keyset inspired by Micro Switch's Halleffect Keyboard that I had the privilege of typing on when I visited the Kono/Input Club office last year. Featuring Cherry's vintage R012345 profile.



When?
November 1st 2019 - December 16th 2019

Where?
US: https://kono.store/products/gmk-ascii
EU: https://mykeyboard.eu/catalogue/category/group-buys/gmk-ascii_214/
CHINA: https://www.zfrontier.com/m/5648
ASIA: https://en.zfrontier.com/products/gmk-ascii
AUSTRALIA: https://dailyclack.com/products/gmk-ascii

Money?
Base Kit: 130 USD / 124 EUR / 962 CNY / 188 AUD
Icon Modifiers: 30 USD / 33 EUR / 193 CNY / 51 AUD
40bit: 50 USD / 47 EUR / 319 CNY / 68 AUD
Spacebars & -keys: 20 USD / 19 EUR / 127 CNY / 33 AUD

Ships?
16-20 Weeks after GB closes

Colors?
Stock colors, 2M on 2B.

Kits?








Pics?
































Comments?
I'm not exactly sure why Micro Switch's keyboard does not feature ASCII sublegends on all alphas, I did complete it using the standard ASCII table. Thanks to constexpr for the idea of putting the remaining commands on other alphas with a 64 character offset.
