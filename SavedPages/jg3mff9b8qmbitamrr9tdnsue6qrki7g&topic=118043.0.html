The Rex60
Pre-order Now | Discord

Supported layouts

Silkscreen Layout Assistance
Detail
Cat

The Rex60 is a GH60 compatible PCB with multi-layout support.

Supported cases include:
  • GH60 tray mount
  • CannonKeys Bakeneko60
  • Tofu60
  • And many other GH60 compatible cases!

Features
  • JST + USB-C support - A new USB-C connector, with fine-pitched through hole leads, can be hand-soldered
  • As many layouts as can be supported. Swiss cheese PCB!
  • Rotary encoder support
  • Bakeneko/gummy-mount cutouts
  • Layout-assisting silkscreen
  • Vial support
  • $29 + shipping

All Rex60s will include a USB-C port. If you would like the USB-C port to be pre-soldered, please select the pre-soldered USB-C option for a $5 fee.

Add a Unified Daughterboard + JST for $5!

Estimated delivery time is 4-8 weeks after the conclusion of the group buy.

Thank you to all of the members of the community that have made this project possible. Shout out to CMG, Suyash, and Zykrah.
