Quote from: Bravado on Thu, 19 August 2021, 11:11:25
Is spacebar the only keycap using the new mold? I'm very concerned about this, for literally all keycaps longer than 1u have wrapping issues in my ePBT sets.

ePBT sets after ePBT Spectrum should be using new revised molds for longer keys like the spacebars. BoW Hangul will 100% be using revised spacebars. As for other keys larger than 1u like 2u, they should be using a different mold as those require reverse dye sub. The are currently no sets to my knowledge that are using new 2u's or 1u. The first samples created for reverse dye sub were just created and sampled a couple days ago.
