GB Closed

SAMPLES



VIEW FULL ALBUM / COMPARISONS



Taking inspiration from the dazzling lights of the big city, this keyset features pops of color set on a strong blue-black base. The colors draw a close parallel to traffic lights, while the novelty icons are derived from classic orientation signage.

Vendors

US - Novelkeys.xyz

EU - Mykeyboard.eu

ASIA - zFrontier for China / for other parts of Asia

OCEANA - Daily Clack


Kits

Base kit



Due to the amazing success of this set, we'll be adding the following extra keys into the Base Kit.



  • Yellow Return - To give a proper mod color accent to this key
  • Teal Arrow Cluster - This gives another option for arrows, as well as provides a much requested accent for those that got the midnight kit
  • "N" Logo Novelty - My personal mark, but also isn't out of place sitting next to the current novelties
  • 2U Shift - This will give proper compatibility to those using the "compressed" 60% layout, as well as some smaller boards
  • 1.25 Tab / 1.75 Return - These keys will help give correct row-profile and legend coverage to some popular sub-60% boards

Midnight kit



Signs kit



Blocks kit




Deskmats

Pulling from the novelties, but keeping it abstract enough to give a nod to the intertwining turnpikes and traffic congestion maps. These mats come in two sizes, 900x400x4mm or 450x400x4mm
