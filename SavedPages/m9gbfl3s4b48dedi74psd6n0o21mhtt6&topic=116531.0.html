DCS Reaper

Death Comes For Us All
DCS Reaper is a set inspired by Death AKA The Grim Reaper.  The Grim Reaper has had many representations in pop culture and literature, but one thing is clear, when Death comes for you, it means your time on this Earthly plane has run out.  I tried to capture the feeling of how Death is often portrayed, coming from the shadows, cloaked in a black cowl, only thing visible are his hands and face, carrying a scythe, knowing he has come to reap the souls.  To capture this feeling I have gone with a dark grey on black monocolored set but with a few bone white novelties (created by OneCreativeMind) that are a stark contrast to the black.  The set is once again in my personal favorite DCS profile and manufactured by Signature Plastics using their double shot abs molds.

I really am excited to share this with everyone now that it is finally in a place where I am happy with the colors and the kitting.  It uses 3 stock colors only, NN for the black, GTA for the grey and WCK for the bone white. 
Manufactured by Signature Plastics in Washington State using Doubleshot ABS in the DCS Profile with the Classic Modified Gorton Legends.

DCS Sleeper uses a Cherry lbased font, this set uses the Gorton font that is traditional to DCS so the font on the renders is what it will look like.

The 10u uses 7u stabs, so technically you can use it on the W1-AT since that supports 7u stabs but the bars and wire that come with the W1-AT that are 10u use 8u stab positioning. By using the 10u bar with 7u stabs, this will support any board with a Tsangan bottom row.
