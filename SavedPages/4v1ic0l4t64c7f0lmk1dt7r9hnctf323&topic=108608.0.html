SA Bliss is a pink and grey bedroom inspired set.
Bliss is back in SA sculpted 1-1-2-3-4-3 profile by SP in ABS double-shot.

Updates
10.21.20 - Order placed with SP.

There is a high likelihood of PBT cherry profile Bliss running this year as well.

Difference between sculpted and uniform.

GB runs September 15th to October 15th 2020
Expected to ship Q3 of 2021.

Vendors
NA: Minterly Studios
CA: Aura Mech
EU: My Keyboard
Asia: iLumkb
Oceania: Daily Clack

❀ Notes ❀

I will be using a lighter grey for this round of Bliss. In the photo below, the grey on top will be the color used this round. If you have Alpaca switches, then you can use that as a reference for what the color is.
