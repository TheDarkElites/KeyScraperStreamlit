[GB] Alps Throwback - Brand New PBT Dye-sub Alps Keycap Set
« on: Wed, 04 July 2018, 03:47:17 »
Hey folks!

Introducing a project which is near and dear to me as a big Alps switch fan: Alps Throwback. I have been working with Melissa at Signature Plastics to explore the idea of a PBT dye-sub keycap set for the last couple months using their 1Z Alps mount (not the same stem mold as Alpine Winter) and PBT to avoid stem breakage and fragility issues that the community experienced with previous double-shot Alps keycap group buys. The keycap set will be made in DCS profile (cylindrical like Cherry profile and with similar sculpt and height though not as thick as GMK keycaps).

One really cool thing you should know about Alps Throwback!

The stabilized keys have Cherry-style inserts, meaning if you combine Alps Throwback with the new XD60 v3.0 with Alps+MX support (disclaimer: I'm also running that group buy at the same time), you can use GMK or other PCB-mount stabilizers (like Zeal) "natively" with vintage Alps switches, which IMO is a pretty killer feature (I've been down the Alps stabilizer rabbit hole for certain builds and it can be pretty painful).

Also, that XD60 v3.0 has split spacebar support, so with Alps Throwback you can have a split spacebar 60% keyboard with GMK stabs and vintage Alps switches which is pretty insane to think about being possible in 2018

AESTHETIC AND INSPIRATION

This is not the flashiest keycap set in the world, no, and I know that the Alps community is small but passionate and tried to strike a balance between fun aesthetics with funky Apple M0110-inspired text-only legends and the blue accents of IBM Industrial Model M with classic go-with-any-case-color beige+grey base colorway and crispy black legends.

Here's are a couple renders by janglad on Modern M0110:




LAYOUT AND KITS

I developed the keycap kits with the community in mind, accommodating the most popular Alps custom layouts such as 60% (Modern M0110/HHKB, Infinity, AEK), 65% (Clueboard), Ortholinear (Planck, Preonic, Nyquist, Let's Split, all the other crazy awesome split Alps+MX boards by keebio) and TKL:



TIMELINE

Alps Throwback GB is currently open until 7/22 on Desk Candy (US) and Mykeyboard (EU):

  • US: https://www.deskcandy.xyz/group-buy/alps-throwback-keycap-set?category=keycaps
  • EU: https://mykeyboard.eu/catalogue/category/group-buys/alps-throwback-kecap-set_89/

PRICING

This is the current pricing and order quantities (as of 7/2/2018) with potential to lower price if we hit quantities of 50 or higher on the kits.

US (Desk Candy)
  • Base: $165
  • Nav/65%/TKL: $50
  • Ortholinear: $35
  • Arrows: $15
  • Split spacebar: $25

EU (Mykeyboard)
  • Base: €165
  • Nav/65%/TKL: €54
  • Ortholinear: €39
  • Arrows: €19
  • Split spacebar: €29



KITS NEEDED FOR KEYBOARDS / LAYOUTS

  • Modern M0110: Base
  • Alps64 with AEK or Infinity Layout: Base
  • KBParadise V60: Base
  • Tenkeyless / 87-key (e.g., KBParadise V80): Base + Nav/65% + Arrows
  • Clueboard: Base + Nav/65% + Arrows
  • Planck / Preonic / Nyquist / Let’s Split: Base + Ortholinear + Arrows
  • XD60 v3.0 with split spacebars: Base + Split spacebar
  • XD60 v3.0 with split spacebars and arrow cluster: Base + Split spacebar + Arrows

We really need the community's support to bring this project to life, so huge thank you to everyone who expressed interest in the interest check and let's have a great group buy! Thanks!!!

For those who intend to use the 6u spacebar from this set or Alpine Winter, Signature Plastics uses different stabilizer peg spacing than (centered or) off-center Cherry/GMK 6u spacebars. This makes it hard to find a 6u stabilizer wire which will work with the Alps Throwback set and unless you are using a custom PCB like me, plate mount Cherry stabs will be a must.

Looking in the Deskthority wiki, it lists the dimensions and TG3 BL-82 under keyboards with 6u spacebars.  The TG3, also known as the cop car keyboard can be found NIB on eBay for under $20 shipped. While it is not optimal to buy a whole keyboard just for the stab wire, they actually have MX Blacks and a full R1 DCS doubleshot BOW keyset. Hope this helps someone.
