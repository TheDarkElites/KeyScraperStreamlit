Features

Left and right interchangeable type-c
Bluetooth Hotswap / Solder PCB (ANSI/ISO)

Design Concept
More

1. More than one typing feel option
(1) PCB Gasket(plateless): bounce; (2) Gasket: a little bounce; (3) Top: stiff

The method to realize it is to use a specially designed middle frame.
Turn over the middle frame to clamp PCB or plate with gaskets, so that we are able to switch between PCB gasket, gasket and top.

For PCB Gasket, the FR4 plate ear needs to remove;
For Gasket Mount/Top Mount, the PCB ear needs to remove (it’s irreversible)
 

(1)PCB Gasket → Gasket (By turning over the middle frame), below shows how it works.

(2)Top mount (The plates have screws holes for top mount)


2. Interchangeable type-c connector and decoration
Someone like left type-c and someone like right one, so we designed a type-c connector that can place left or right as you like.

Appearance Design

As the studio manager's name is Chain Smoker, and we want to make the keyboard more distinctive, so we add some cigarette elements to it.
Although we used some cigarette elements, we do not encourage smoking, so we engraved “ Smoking is harmful to health” inside the bottom case for reminding.




PCB Features
Hotswap PCB with various layouts: TKL/WKL, both support ANSI / ISO
Leave the solder place for knob on F13
Upgrade to black core Bluetooth Hotswap/ Solder PCB, which is better in quality than the ordinary one
