When I was entering the hobby I was looking for a set like this but it didn’t exist in double-shot abs for mx. This is great, but I think it’s worth considering who the audience is. Clearly you’d go after this if you’re a mac user, but for me it’s missing all the modern functions to be useful. I understand what this is designed after, and I appreciate the new legends on alphas, but not having the modern F-row macros is such a miss. I’m talking about brightness, volume and media controls. It’s a modern standard, even older boards from powermac era had these, and it’s extremely convenient.

In fact, as someone who works in professional software 100% of the time (graphic design, CAD, CAM, DAW and NLEs) I still use the f-row for media controls much more than I do for app features.

A set of Novelties as extra would have been fine too.

Such a shame, such a great set otherwise.

We could have offered Media kit but keep in mind that this project required brand new molds being made, which is expensive. Even if it's inspired by the AEKII, I had to make the choice of reason, the one that would suit the majority of potential buyers, which is to offer regular F1-F13 keys. That "modern" F-row could still be something we'd offer in a future run though (depending on how it does). The same applies to Japanese sub legends actually, dedicated 40% kit, and so on. If it were up to me, and production costs were insignificant, of course I'd offer as many options (kits) as possible. To have a keycap set featuring only new molds at such a price is already sensational, to be honest.

Thank you for your comment, though. I'm glad you still like the set as it is.
