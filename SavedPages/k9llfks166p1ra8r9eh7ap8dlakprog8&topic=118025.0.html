DCS Dark Sky

GB August 5-19 2022

This project is inspired by my all time favorite colorway, Soware. I had always wondered what a dark version of the original DCS soware would look like, so I decided to make it a reality.

The Colors


Alphas: GPA (same as original Soware mods)
Mods: GQM (not used in original Soware)
Legends: BFQ (original Soware uses BBQ and BBJ, I prefer using only one color and this splits the difference)
Red: RDD (original soware uses RR)
Blue: BDH (same as original Soware)
Yellow: YCH (original Soware uses YY)
Green: VCS (original uses VCM, I prefer this green)
Orange: OBH (not used in original Soware)

No color matching needs to be done with SP before this set begins production, as it only uses stock colors. HIBI will be sent physical color chips for color matching once the sales period concludes.

NK87 by Novelkeys




Vendors

US: Novelkeys (145 USD)
CA: Desk Hero (190 CAD)
UK: Prototypist (123.33 GBP)
EU: Oblotzky (159 EUR)
Asia: zFrontier (1099 CNY)  (EN link (165USD))
AU: Daily Clack (229 AUD)
SEA: IllumKB (215 SGD)

Lead time will be around a year, so Q3 2023.

HIBI

More


One base kit. One HIBI.

As far as the kitting goes, I wanted to have physical support for as many layouts as possible, meaning that the vast majority of people will be able to buy this set and have it fit on their keyboard, even if the legends aren't quite right.
My only keyboard regret is not buying the tsangan kit for the DCS Smallfry set and now I can't use it on most of my boards. I don't want that to happen to anyone who buys this set.
