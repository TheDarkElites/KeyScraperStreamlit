Quote from: smithyithy on Mon, 03 April 2017, 12:56:04
Could I use this to effectively build a 'custom' 60% board using Alps / Matias switches? So the 'AEK' term simply refers to the layout / key sizes?

Yes.  You'd need a universal 60% case (preferably metal to avoid pinging) and a PCB (which can be bought from other users or Hasu in the ALPS64 thread), as well as donor switches and correctly sized/stemmed keycaps.
