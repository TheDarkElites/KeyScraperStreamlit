Introduce:

JTK Classic FC(designed by Buger) is heavily inspired by the beloved Nintendo Family Computer (Famicom) in the 80s. The red, beige, and black color combo is the original colors from the Japan version from the eastern market, while the NES distributed in the west is in gray and black.

This is the first ever Cherry profile keycap set featuring tripleshot ABS latin and hiragana legends!

JTK will be introducing some "surprise mechanics" (lul EA) for this GB

Blind Box
JTK will be including a blind box (random novelty keys) per Novelty kit
@500 Base, there will be 5 blind box novelties in total, and each order will get 3
@600 Base, there will be 6 blind box novelties in total, and each order will get 3
@700 Base, there will be 7 blind box novelties in total, and each order will get 4
and so on...
 first 3 variants are shown in the Novelties kit, the remaining ones will remain a mystery
Scratch Cards
Each Novelties kit will also come with a scratch card, which may or may not include random JTK keycap sets or a JTK Keyboard https://geekhack.org/index.php?topic=103121.0, as shown in the IC post in the prize pool.
Fulfillment for the JTK keycap sets and Keyboard will be handled by JTK

COMPATIBILITY
ISO / ANSI layout (60% / 80% / 100%)
Cherry G80-3000, 3494
Leopold FC660M / 980M
WinKeyLess layout
1800 WinKeyLess layout
Customized 60% / 75% / 80% / 100% layouts
HHKB layout (Duck Viper / Eagle)
Split Spacebar support with Spacebars kit

SPECS
Designed by Buger
Cherry profile
High quality 1.5mm doubleshot and tripleshot ABS plastic
Packaging: sorting trays
Manufactured by JTK
Made in China

TIMEFRAME
Preorder: Feb.12.2020 - Mar.15.2020
Estimated Shipping Time: Sep.2020
