Hi everyone! Many of you have probably seen teasers of this set from either our Discord or Instagram, but it's finally happening!

ePBT Classic Retro Cyrillic!








We decided to go with one large base kit for this set. We kept the same original kit from the GMK version of this set, which optimized compatibility for several layouts.


Vendors:
USA: Originative Co.
CA: ApexKeyboards
EU: protoTypist.net
SEA: Monokei.co
AU: DailyClack

Pricing:
USA: $99 USD
CA: $140 CAD
EU: £100 (~€111)
SEA: $152 SGD
AU: $160 AUD

Runs from 6/12 to 7/17

Estimated Shipping Date: December 2020


We decided to drop this set as a surprise for several different reasons, but we hope you guys like it!


Also, feel free to join our Discord server if anyone has any questions about this set!


Special thanks to all of the vendors who have helped out on this project (especially on such short notice), and my friend who owns the GMK version of this set that inspired me to create this project
