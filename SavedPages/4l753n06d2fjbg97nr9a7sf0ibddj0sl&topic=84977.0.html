[GB] DSA LightCycle Full Set (CANCELLED)
« on: Sat, 01 October 2016, 12:36:12 »

This set has a Tron theme and features some beautiful novelties.  There are three kits, base ($80), compatibility/international ($50), and numpad ($20)
Both MX and Alps/Matias stem mounts are available.
A wide range of layouts are supported with this set including ergo dox, MiniVan, JD40, JD45, Sebright, Minorca, Whitefox, HHKB, 1800, winkeyless, etc.

Group buy closes on the 17th of October 2016
