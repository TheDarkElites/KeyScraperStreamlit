NEON DYNASTY
An Official Magic the Gathering™ Keycap Set



A keycap set inspired by Kamigawa: Neon Dynasty.

Inspiration:
This set is designed around MTG release, Kamigawa Neon Dynasty. This cherry profile keycap set uses
the official MTG font and futuristic Hiragana sub-legends. This keycap set features strong vibrant colors
and a futuristic Japanese influence to inspire all of your magical cyberpunk dreams.

After our overwhelmingly positive IC response, we've been working tirelessly to bring this set to life.
We have been working with the team at Wizards of the Coast, Color matching with Infinikey, and
making sure we have this set ready for prime time.

The All-In-One Basekit is priced at $149 USD.
Available Exclusively at:
Clackeys.com
Due to licensing restrictions we are not able to distribute this set outside of the US and CA.
GB runs from Jul 25, 2022 to August 25, 2022
We are expecting delivery 6 months from end of GB.


The Set

All-In-One Kit

This All-In-One kit represents the changes we made after we received feedback from
the prototype set. The most important change was adding the new accent Spacebars.
