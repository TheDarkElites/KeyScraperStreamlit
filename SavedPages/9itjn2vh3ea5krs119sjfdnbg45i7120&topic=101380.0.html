[GB] BIOI G60BLE 60% PCB
« on: Fri, 28 June 2019, 08:49:25 »

Hello community!

Introduce you another bluetooth 60% custom PCB by Scott Wei.
BIOI G60BLE PCB was originated from Scott's G60 keyboard kit. The bluetooth PCB was combined with a mother board and a daughter. After some modifications, it become a bluetooth PCB with general specifications suitable for stock 60% keyboard parts.

Full Album: https://imgur.com/a/KnU0SdR

Content:
  • PCB
  • Two SH to PH adapter cables with inverted sequences
  • SMD LED for a charging indicator position
Features:
  • QMK firmware
  • USB C connector
  • BT4.0, available on most devices and systems like Win10, iPad and iPhone. Super power saving.
  • 1.0mm SH connector for battery socket
  • VIA Configurator supported (Thanks for help from Wilba and Olivia)
BIOI G60BLE PCB will come with two PCB versions in this groupbuy. A default one which you have to solder switches, and a hotswap one for easier installation.
Default (55USD):
  • South facing LED position
  • Multi layout support
Hotswap (60USD):
  • North facing LED position
  • Kailh hotswap sockets
  • 7u Spacebar layout only
GB link: https://play-keyboard.store/products/copy-of-play-keyboard60-pcb
GB Period: 2019.6.24 ~ 2019.7.10
Scott Wei's Instagram: https://www.instagram.com/scott_wei

PS.
  • If you have joined G60 custom keyboard kit groupbuy and you want to purchase this item, please contact scot.wei@gmail.com directly.
  • If you need assembly service for getting a full built board with this PCB, please contact our customer service.
Please feel free to leave comments if you have any question to this product.
Thanks!
