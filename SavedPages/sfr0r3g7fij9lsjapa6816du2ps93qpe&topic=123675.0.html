Hello and welcome to my debut set: GMK Cosmos.
A black on black set using GMK’s custom, special-effects plastics.
Cosmos is black with iridescent dust.

GMK COSMOS FEATURES

  • Cherry profile, ABS plastic, thick doubleshot keycaps manufactured by GMK using one of GMK's new custom, special effects plastics.
  • Colors: Cosmos base with CR (Black) legends. Accent keys are CR base with Cosmos legends.
  • Colors: Cosmos is a custom, special-effects plastic that is black with iridescent specks of color.
  • OG Cherry-style Icon-Text modifiers.
  • Legends on LED keycaps are printed. Windows are CR Transparent 69% for a smoky look.
  • No novelties kit. Instead, please consider artisan collabs! Announcement on those soon.
