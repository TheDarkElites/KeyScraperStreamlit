RubySea 1800

Instant Keyboard Quoter


RubySea 1800 and Instant Keyboard Quoter



Background

Hello everyone! Jack from Pikatea. We recently launched a new service for providing custom keyboards with the layout of your choosing for low MOQ and low prices. I'm able to pull this off by using parametric design, code automation and 3D printing to bring the cost of manufacturing way down. All you have to do is input the "Keyboard layout editor" raw data into the website and it instantly gives you an estimate! It's not intended to be ridged or strict but more of a tool that gives you an idea of what it might cost to have the keyboard made. There's a comment box to make special requests and I factor those in when providing the final official quote! I encourage you to try out a layout and see what you think! All feedback is welcome and we have a form on the site for it!

Enough about the site! How about the keyboard?

The RubySea 1800 is a "test" or a proof of concept in the models and code I've written. It's a simple 1800 layout keyboard kit with a display and a knob. It uses FR4 for the main material with an optional 3D printed case "upgrade" that surrounds it. It's nothing fancy but it's a fun kit for those that like to build every aspect of the keyboard!

Specs

* FR4 through-hole kit

* OLED
