Hi this is a bit off-topic for this thread, but it is very much not the same. Some countries (Denmark) has a 21.5€ flat fee in addition to VAT (25%). For this set with free shipping, it still isn't too much of a difference, but:
- If there should be any keys missing/defective, and a single key should be sent free of charge, it is still subject to customs. This might mean that you have to pay 21.5€ handling fee to pay 25% of a 1€ marked value, totalling at 21.75€ fee (yes people actually gets this bill, it's not made up).
- If the set is to be returned, having the VAT refunded is nearly impossible at such low amount, afaik
- If the vendor has a policy where the full set has to be returned in order for a new to be sent out, this will most likely result in VAT + the fee being paid twice

If Drop offered pre-paid VAT, it would be easier to order from them without these additional costs.

(Also sending goods from Germany to the US, only to send them back to Germany and their neighbouring countries seems not-so energy-efficient)
(Also small businesses aren't bad. There is a lot of criticism to be dealt out among a lot of vendors that have folded, but I won't let that dictate me to not trust anyone who isn't backed by a massive company)
(I'm not complaining about Drop in general, just a reply to this message)
