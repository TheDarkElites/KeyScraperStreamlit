SUBJECT: GMK KEYCAP SET █████████

You are being assigned to a highly classified case that began in July of the year 2019. ██████████████████████████
███████████████████████████████████████████████████████████ Certain parts of this document
have been redacted because some of the information is above your clearance level. The main information you need to know to get the job done
is that this set will be use doubleshot legends with pad printed redactions on top of the doubleshot legends.
GMK stock colors will be used.

One advantage of a project that has this much information redacted, is that it can cover nearly every sized
keyboard layout in the base kit. Each redaction was carefully chosen so that the correct amount of each sized
keys on each row that are needed for more obscure layouts, which are usually not covered in base kits. In the Clearance Kit Flat Lay Image below,
you will find a list of how many of each key are redacted on each row. This is based on our Obscure kits we usually run. So there should be
the same coverage that the Obscure kit gives from our past groupbuys. ███████████████████████████████████████
