« on: Wed, 03 April 2019, 14:52:08 »
Here we go again. GMK Eclipse is now available on Massdrop.
TO THE DROP PAGE
The drop will last 27 days.

As the name of this set tells the theme of this keyset is Eclipse. Eclipse is an well known astronomical event (no its not IDE based), thus the keyset has dark appearance. The lighter modifiers do symbolize the typical light on the edge of an astronomical body which takes part in the eclipse event.
For more information see GMK Eclipse interest check.

Sheet with MOQ and pricing (will be updated as soon sheet on the drop page gets it updates.)

Sale charts by Dvorcol.



The previous Accent & Novelty kit was quite expensive and i decided to split this kit in its parts and make it more affordable that way.
Overall this is my attempt to provide more affordable set and if it fruits i will continue similar pattern in my future projects

Thanks for the interest and hopefully support

I got simple rule for when to use icon or icon + word modifiers. Very distinct contrast colors are getting only icon mods. Less contrast colors are getting icon + word modifiers. You can see that rule of mine applied in GMK Carbon R2 drop. Very conrasty Bone has only icon  mods where less contrasty Carbon (orange on gray) has icon + word mods.

In case of Eclipse of course the mods need to be icon + word. For only Icon the mods need to have color Yuri light blue on N9. But thats GMK Skydata.
