Update 8/22

Hi everyone! It's time for another update.

Unfortunately, today's update isn't as cheery as the last one. All the machined parts have been polished and prepared for anodizing, but we've run into some hurdles with that process.

Anodizing is a very painstaking process, and minor changes in environment can result in drastic changes in the final product. While doing some initial tests on the anodizing bath, we weren't happy with the quality of the final result, and so we've put the anodizing on hold while we can assess the problem and put a solution into place.

While this delays getting the boards into your hands, we aren't comfortable rushing these boards and sacrificing quality. We apologize for these delays, and we hope for your understanding.

As always, stay safe everyone, and we'll be in touch!
