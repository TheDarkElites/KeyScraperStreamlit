GMK Boneyard


Group Buy is active from June 5th 2020 to July 3d 2020
Est. Shipping Dates will be between February and April 2021

Hello everyone and welcome to Group Buy for GMK Boneyard. GMK Boneyard was an optional colorway offered in GMK Carbon R2 in 2018 and it was quite expensive to get, if you consider that you had to get a base kit and the boneyard alphas. So in order to make it more accessible i decided to run it as standalone set with more competitive pricing
Additionally it offers some unique novelty variations which were not available in GMK Carbon R2. You should also keep in mind that those novelties can be combined to some extent with previous version of GMK Carbon and partially with GMK Oblivion (Oblvion mods base color is the same as used in Carbon). So all in all there is some room for mix and match available

Link to GMK Boneyard Interest Check thread.

Vendors:
  • EU: Oblotzky Industries
  • NA: NovelKeys
  • Canada: Deskhero
  • South East Asia: Monokai store
  • China: CN:ZFrontier/EN:ZFrontier
  • Australia: Daily Clack

Base kit

The base kit offers only the necessary key for 60%, 65%, TKL, WKL, 100% and 1800er layouts. I do expect the base kit to get very competitive pricing.

40% kit

This kit is also designed to be affordable. Thus it includes only necessary keys for most 40% layouts and it contains all currently available spacebars for layouts with split spacebars. Additionally it includes extra B key for layouts like TGR Alice.

Novelties kit

Most of all my sets do offer some sort of novelties and Boneyard is not an exception. The theme is here obvious. Its about Pirates/Buccaneers which adds some adventurous touch to it.

Domino kit

This kind of novelties was introduced in GMK Carbon R2/SP SA Carbon R2 and here it is available in the Boneyard colorway which sets it apart from the previous runs. Anyway if you want to display the number keys on your keyboard in a different way then this is the kit you should get
