SA METAVERSE



ENTER THE METAVERSE!

Inspired by the latest installment in the expansive Persona video-game series—Persona 5—we’ve resurrected a keycap set that’s sure to steal your heart. It’s called SA Metaverse, and it’s an updated revival of 2018's popular GMK Metaverse Keycap Set that captivated Persona fans across the globe. Staying true to the original set’s clean, cartoonish style and otherworldly inspiration—the Metaverse: a supernatural realm born from humanity's subconscious desires—our set is sculpted in SA profile to give fans a new feel.

Designs and updates to the kits were adjusted to have a more compatible fit to the SA profile; that is produced by trusted manufacturer Signature Plastics for exceptional build quality down to the finest detail. For fade-free performance, each kit is made from durable ABS with doubleshot legends.

It’s time to enter the Metaverse; your keys are right here.






Purchase between 12/2/21 - 1/2/22 for a launch discount!




Estimated ship date is Aug 9, 2023 PT.

Q&A for this set based on the discussion on drop:

- Why is this running in SA?
   - The designer likes SA and wanted to run it in this profile, it is not running in SA because of any issues with running an R3 in GMK.

- Did you get Krelbit's permission to run this?
   - Krelbit did not design Metaverse...

- Will there be an R3 of the GMK version of this set?
   - Yes, I talk to Dead Encryption daily and have been told many times by him that there will be an R3 of the GMK set at a later date.

- What about IP issues, isn't that why it's not running in GMK again?
   - GMK has never said that they would not run any more sets that were inspired by or based off IP, per the post that is spreading around from Krelbit's server Krelbit will not be running any more sets inspired by or based off IP unless explicit written permission is given by the IP holder(s). The then goes on to say that any vendor will be liable for not only themselves, but GMK and all of the regional vendors if any IP related legal action occurs. Meaning that GMK puts full responsibility on the lead vendor if there are any legal actions when it comes to IP issues.
