Hi, I guess I'll try to add my 2 cents to the thread.
First of all, I think the set look great, and ngl, I'm sad seeing it cancel.

We probably don't know what's going on behind the scenes so I'll just make assumptions.

1- The name. Not a problem by any means it's different enough from the name of the band, names are very regulated but here, I'd be amazed if someone tell me this is actually "close enough" to the OG one. I doubt any judge would think the name is problematic here.
2- The color scheme. You can own some colors (Tiffany blue, the blackest black, space gray...), and in some situation yes, you can own some color schemes but it's so complicated that I won't go into details about it to make things easier, but that dosent apply in this situation.
3. The overall "inspiration" of the set. Some people might think that the set having KR legends, inspired by this K-Pop brand and having the same color scheme is enough to claim that it's a violation against IP rules is.... not acceptable. You're allowed to get inspiration from anyone as long as you justify of a some degree of creativity to make your creation different. Which leads to my last point.
4. The noveltie kit. The rest was perfectly fine if everything had stop there but here is the real deal. Having such a kit that have text novelties that remind the songs of the artists, novelties that replicate their faces etc. Here we have too many clues of a violation of the IP of the K-Pop Band and I definetly agree on that. (Well the deskmat aswell showing the girls should be remove too in that sense).

The solution would simply be to remove the noveltie kit and the problematic deskmat. That way the set will be different enough, and just "an inspiration" and there is no way a trial could even start with such few elements.

That said, I hope this set will comeback because there is no reason it should be canceled beside the 2 that I just mentionned.

Also note that I made things simple for the sake of everyone and that I'm specialized in Fr/EU IP so things can vary but that shouldnt affect this particular case by any means.

If someone disagree with that i'd be curious to hear them.
