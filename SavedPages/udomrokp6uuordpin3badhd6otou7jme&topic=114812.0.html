Inspiration and Colors

More
DSS All Hallows’ Eve is a pick your own kitting style doubleshot DSS (scuplted) ABS set from SP inspired by the many childhood memories I had growing up during Halloween. The set uses the popular trashcon modifiers as well as a uniquely designed set of novelties.

This set will be using stock Signature Plastics colors:
Cream (WCK) alphas/mods
Black (NN) legends/accents
Orange, Blue, and YCA (OAS, BBQ, & YCA) novelties/accents




Allow me to elaborate on the inspiration of this set. As a kid, Halloween was always an escape. An escape to a world of fantasy where ghosts and ghouls existed and roamed free. Think Scooby-doo Ghoul School type of situation. Halloween was, and still is, the one holiday where all problems no matter the magnitude disappear and you get to enjoy time with friends and family. Not to mention all the free candy! I wanted to create a set that shared this happiness in a creative and playful manner. The set also draws inspirations from my other expensive hobby, sneakers. Some of the colors for this set, mainly the blue, come from the OFF WHITE Nike All Hallows' Eve blazer. The set and the shoe also share a name for their striking similarities!
