[GB] XDA Oblique (complete)
« on: Thu, 05 July 2018, 11:51:32 »


IC thread link

XDA Oblique is a keyset inspired by the keycaps of the AEK, M0116, AEKII, and similar Apple keyboards. The font used in this keyset is Oswald Light and is angled at 17°. This is a very close to Univers 57 Condensed Oblique, the font used on the AEK. I have modified/resized some of the symbols to match the AEK. Even Univers 57 isn't an exact match to the AEK's font though; it is slanted at a smaller angle and several of the symbols are different. The colour will be Pantone Cool Grey 2 U, which matches Apple's colour very well.

Quote from: ultrasiq on Mon, 16 July 2018, 16:18:10
Sorry for the noob question, but which sets(s) would I need to fit my Vortex Race 3? I know it has some funky key sizes.

Either set of alphas, either set of base mods, the compatibility kit for 1u bottom row keys and 1.5u delete.

The 1.5u esc isn't included (unless you buy the ergodox kit for just one key) because it's pretty extremely non-standard. You can get the blanks kit to fill it with a blank or just use any leftover 1.5u key if you don't mind an incorrect legend.
