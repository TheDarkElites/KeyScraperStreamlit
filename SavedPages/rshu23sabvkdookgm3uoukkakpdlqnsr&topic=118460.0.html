INTRO

Hello, Manu from ELOQUENTCLICKS here (my actual name is Manu, I’m not a manufacturer, please stop). I am introducing my first set of keycaps, DMK ЯUBBER, a set that seeks to show what would have happened if Rubber Hose drawing style had been invented in the former Soviet Union.

Manufacturer: Domikey
Material: Doubleshot ABS
