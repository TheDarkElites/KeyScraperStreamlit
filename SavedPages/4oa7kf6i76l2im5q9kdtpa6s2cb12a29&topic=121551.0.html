KeyKobo Miami Nights

Discord | Website

Big cities never sleep, so why should your desk? Miami Nights is a classic set with quite the lore. Stunning on dark boards, while discrete on loud ones, this set is a must have for any collection.  

Kitting:


A large base kit provides extensive physical support for most common layouts and some legend preference options too.


A NorDeUK kit for language compatibility in a variety of regions.
* Please note only Keygem, Prototypist and ZFrontier will be offering the NorDeUK kit.

Colours:

Colour matched to GMK’s CR/MG1/TU2 colours. See samples below.

A small mistake has been found in the base kit render files regarding alignment of the icon modifiers in Miami Nights. Thank you to a member of my discord server for pointing out the discrepancy here:

Unfortunately the molds that KKB offer for icon modifiers are of the centred variety for the Shift keys and Purse caps. This is in line with the specifications provided to Keykobo for the manufacturing of another set by a large vendor.  As such all of the keys will be center aligned vertically as this is the only option available.

I'm sorry if this is disappointing to you and I really hope it's not a dealbreaker. Thank you for your support with this set so far. I think this change is quite minor but I understand small things like this are important in the community so I wish to be as transparent as possible.

I've updated the OP with a fixed kitting render, this will be updated by all vendors to match soon. In the morning I will be taking the time to update board renders.

Regarding the previous change to modifiers for KKB Miami Nights, this has been reverted. After discussions with KeyKobo we have confirmed that new molds will be made to support non "Dixie" style modifiers. This also means that with the success of this GB modifiers of this type will be available from KeyKobo.

This means that the original renders from the beginning of the GB are now accurate. I'm once again updating all material to reflect this. It is likely we will extend the GB by an extra week to allow for more orders to be placed in light of this change.

Tldr; the shift and lock modifiers will not be centred (Dixie style), they will be in line with traditional Cherry icon modifier placement.

I have updated the renders in the OP with the correct images
