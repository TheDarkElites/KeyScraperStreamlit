Alexoto's Build Stream VOD

SPEC



► Price: $470 (Regular) / $535 (collaboration units)
► Form Factor: 70% FRL-TKL with WK/HHKB
► Typing angle: 9°
► Front height: 19.3mm
► PCB: 1.2mm solderable (does not support hot-swap), Type-C with daughter board. PCB supports QMK and VIA, designed and manufactured by Astro
► Mounting: Gasket mount ( Aluminum plate with extra leaf spring design)
► Plate: Default plate option is Aluminum, and optional PC plate
► PCB Layouts: ANSI and ISO (Plate sold separately) with WK/HHKB, split Backspace, split right shift

WHAT‘S IN THE PACKAGE

► Keyboard Kit(Top/Middle/Bottom Case, Plate, PVD Badge, Weight) x 1
► 1.2MM PCB and daughter board x 1
► 0.5MM PE Foam x 1
► 3.5MM PORON plate foam x 1
► C shaped gasket x 10
► 1.5MM PORON bottom case foam x 1
► LED Diffuser x 1
► LED Reflector x 1
► Stabilizer Shims x 10

► Bumpons x 2
► Instruction x 1
