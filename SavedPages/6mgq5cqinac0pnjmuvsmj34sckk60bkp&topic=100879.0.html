Hey all!

I'm so excited to announce that my first GB has gone live on Novelkeys!

US: https://novelkeys.xyz/collections/group-buys/products/sa-fluffy-clouds-gb
EU: https://mykeyboard.eu/catalogue/sa-puffy-clouds-ortho-base_1018/
https://mykeyboard.eu/catalogue/sa-puffy-clouds-ergo-kit_1017/



It's an ortho/ergo focused SA set that I've been working on for a long time. It started with a desire to match a Cyan Shishi that I'm totally in love with, so I tried a bunch of keyset combos but could never really capture the colors. So my friend encouraged me to set up a group buy and make it happen!

This is the just beginning of a crazy fun journey and I'm excited to be taking it with you all! Thank you to everyone who has offered feedback on the set!

---

This keycap set will be run through SP. It is double shot ABS. Color codes are WFK and BBQ. Designed by vosechu, this set is perfect for Ortho and Ergo users. This will be uniform R3 SA keycaps.

Two kits to choose from. Ortho base and Ergo Add-on.

MOQ for Ortho base is 100 and MOQ for Ergo add-on is 50.

GB will run until June 22, 2019
These keycaps are estimated to ship in November, 2019.
