Numbers Update

We've hit 45k units out of 54k MOQ so almost there!
The minimum MOQ is 24k units, so the set can be made no matter what, but we have priced at the second tier at 54k which is why I have stated the 54k goal.

Some more info about KAT MOQ for those not familiar:

KAT MOQ has 3 tiers, 24k (minimum), 54k, and 120k total units. I just learned this last night but a unit = keycap unit NOT a singular keycap, so 2u keycaps count for 2 units.

The way the kit MOQ works is "suggested numbers" to hit the number of required units. I will be providing some numbers to amazing community member dvorcal so he can make some pretty graphs at the end of the week But those kit numbers do not necessary have to be hit for those kits to be made.

Quote from: minhhoang3107 on Sat, 09 November 2019, 10:22:10
Does the 60s/TKL Mods support TGR Alice layout (the split spacebars)?

Short answer: Yes.

Long answer: Kinda.

If the left half of your alice has a 2.25u space and 1u mod, then the only thing you'll be missing is a second B, which you can fill out with another R1 key (up arrow, end, page down).

If you have a 2u space and 1.25u mod, then you'll need to use the 2u shift key which will be the wrong colour for the row.
